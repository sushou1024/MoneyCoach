# Money Coach Web Frontend

## Design Notes
- Display font: `Libre Baskerville` for refined, readable headlines and logo.
- Body font: `Spline Sans Mono` for readable monospaced copy.
- The Method section uses an editorial split with a vertical rail and a text matrix band.
- Badge colors are defined by modifier classes:
  - `badge-high`, `badge-medium`, `badge-severe`
  - `badge-armed`, `badge-ready`, `badge-queued`
  - `badge-aligned`, `badge-rebalanced`, `badge-active`

## Local Development

```bash
npm install
npm run dev
```

Health check: `GET /healthz`
